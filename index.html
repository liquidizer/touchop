Touchop

Touchable operators

Touchop is a mathematical puzzle game that asks you to set up your own formulas, such that a given objective is met. Watch a video on Youtube

Levels

You need an HTML5+SVG capable browser to play this (e.g. Firefox, Chome, Safari, IE11?)
On mobile devices please use Opera, Safari or Firefox. Android's default browser does not support SVG.
